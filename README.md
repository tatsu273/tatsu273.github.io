# 勤怠管理システム

geminiを用いたフロントエンド言語の学習成果として作成したサイトです。簡易的な勤怠管理ができます。

## 概要

ブラウザ上で従業員の管理や勤務時間の記録、勤務記録の確認などを行うことができるシステムです。データはブラウザ上のローカルストレージに保存されます。

## 主な機能

- 出退勤の打刻: 登録された従業員の出勤・退勤の記録が可能。
- 勤務記録の確認: 従業員ごとの勤務情報の確認ができる。前後一年の確認が可能。
- 勤務記録の手動入力: 勤務記録の手打ち入力が可能。
- 勤務記録の入力履歴の確認: 登録されている勤務情報の履歴の確認が可能。記録が1万件を超えた場合は古い記録から削除される。
- 従業員管理: 新規従業員の登録と編集・削除が行える。最大で999人までの登録が可能。
- 有給申請: 登録された従業員の有給申請が可能。もともと休日の場合や申請日以前の申請はできない。
- 設定: ホーム画面に表示されるショートカットボタンの編集、データの初期化が可能。
